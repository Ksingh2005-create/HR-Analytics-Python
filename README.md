# HR Analytics Using Python

# Project Overview

The HR Analytics Using Python project is an end-to-end data analytics project designed to analyze employee data and uncover meaningful workforce insights. The project focuses on identifying the key factors affecting employee attrition, job satisfaction, employee performance, and overall workforce productivity.

Using Python and its data analysis libraries, the project demonstrates the complete data analytics workflow, starting from data cleaning and preprocessing to exploratory data analysis (EDA), feature engineering, KPI analysis, visualization, and business recommendations.

The primary objective of this project is to help HR professionals make data-driven decisions that improve employee retention, optimize workforce management, and support organizational growth.

---

# 🎯 Project Objectives

The objectives of this project are:

- Analyze employee demographic information.
- Understand employee distribution across departments.
- Identify the major factors contributing to employee attrition.
- Analyze employee satisfaction and work-life balance.
- Study the relationship between employee experience and salary.
- Evaluate employee performance across different departments.
- Calculate important HR KPIs.
- Generate business insights to improve employee retention.
- Present findings through professional data visualizations.

# 📈 Key Business Insights

The analysis provided several valuable insights, including:

- Departments with comparatively higher employee attrition were identified.
- Employees working overtime showed a higher tendency to leave the organization.
- Monthly income generally increased with total working experience.
- Employees reporting higher job satisfaction tended to have lower attrition.
- Work-life balance played an important role in employee retention.
- Department-wise performance differences were identified.
- Workforce KPIs highlighted opportunities for improving employee engagement and retention.

---

# 💡 Business Recommendations

Based on the analysis, the following recommendations were made:

- Reduce excessive employee overtime to minimize burnout.
- Improve employee engagement in departments with higher attrition.
- Conduct regular employee satisfaction surveys.
- Strengthen career development and training programs.
- Review salary structures for lower-income employee groups.
- Promote better work-life balance initiatives.
- Improve managerial support through leadership development programs.
- Monitor HR KPIs regularly for proactive decision-making.
